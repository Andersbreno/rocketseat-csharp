# 💻 Introdução a C# e .NET - Rocketseat

Este projeto foi desenvolvido durante formação introdutória em C# e .NET, com foco em fundamentos de backend.

## 🧠 Conteúdos abordados

- Fundamentos da linguagem C#
- Programação Orientada a Objetos (POO)
- Criação de APIs
- Validação de dados
- Entity Framework
- Integração com banco de dados

## 🎯 Objetivo

Compreender os fundamentos do desenvolvimento backend utilizando o ecossistema .NET e boas práticas de estruturação de aplicações.

## 📌 Contexto

Projeto desenvolvido em paralelo com o curso, priorizando a experiência prática e a aplicação dos conhecimentos adquiridos.

---

## 📈 Resultado

Implementação prática dos princípios fundamentais do backend, incluindo a criação de APIs bem estruturadas e o tratamento de dados.
